# Gobler Documentation

Welcome to the Gobler documentation.

Gobler is a tax-aware personal wealth agent built on Strands Agents + Amazon Bedrock
AgentCore, on top of AWS's Fullstack AgentCore Solution Template (FAST).

Start with the top-level `README.md` for the architecture and deploy steps. The deep-dive
guides in this folder (Gateway, Cedar Policy, Memory, Identity, Runtime auth, Streaming,
Code Interpreter, Deployment) are inherited from FAST and explain the underlying platform
mechanics that Gobler builds on.
